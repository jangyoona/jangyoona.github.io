# 서비스 소개
> - 해당 프로젝트는 보육원의 아이들과 시니어를 매칭
> - 지역 사회의 연대와 참여 증진을 기대
> - 봉사자와 활동의 매칭을 통해 사회적 유대감을 높이고자 하는 목적
> - 노인과 어린이 간의 상호 작용을 통해 사회적 책임을 다 하는
  긍정적인 문화 확산을 기대
